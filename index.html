About Me

Since beginning my journey as a developer, in late 2018, I've completed The Modern Python 3 Bootcamp and The Python Mega Course. At the moment I'm working on completing The Ultimate MySQL Bootcamp . I've also recently registered for C# Intermediate.

Throughout building my projects (shameless plug) I've also learnt quite a lot of HTML, CSS, Bootstrap and MySQL. My current position, as a developer at iKind Media, has allowed me to learn a great deal about working with Wordpress.

When I'm not coding you can find me adding to my blog, surfing, scrolling through Reddit, drawing on my Wacom, searching for Durban's best burger or watching series, most likely with some sort of tech relevance . If you're starting to think I'm slightly obsessed with everything code related you're not entirely wrong.
